## Running The App

1. Untuk menjalankan project gunakan perintah :

 ```
 gradlew run 
 ```
2. Untuk menggerakkan tiles pada aplikasi, klik menggunakan mouse pada tiles yang bersebelahan dengan tiles yang kosong
